## ✅ 실습 내용 및 진척도 (LAB Tasks)

총 12가지 주제를 통해 Rust의 동적 메모리/자료구조 활용부터 UNIX 시스템 프로그래밍의 핵심 요소(파일, 디렉토리, 프로세스, 시그널, IPC)까지 심도 있게 다루었습니다.

|  번호  |  주제  |  상태  |  비고/링크  |
|  :---  |  :---  |  :---  |  :---  |
|  **Q1**  | Lab2 프로젝트 GitHub 저장소 생성 및 관리 | ✅ |
|  **Q2**  | Rust(Vec<T>)를 이용한 동적 행렬 덧셈 프로그램 구현 | ✅ | `Vec<T>`를 활용한 동적 크기 배열 관리 학습 |
|  **Q3**  | Rust 전화번호부 프로그램 구현 (HashMap 자료형 활용) |  | `HashMap`을 사용한 키-값 데이터 처리 학습 |
|  **Q4**  | 파일 및 디렉토리 관련 함수 활용 연습 |  | `open`, `read`, `write`, `stat`, `opendir` 등 시스템 호출 연습 |
|  **Q5**  | 재귀적 디렉토리 탐색 프로그램 구현 (ls -R) |  | **재귀(Recursion)**와 디렉토리 탐색(`readdir`) 로직 구현 |
|  **Q6**  | 타자 연습 프로그램 구현 (오타, 분당 타자수 측정) |  | 터미널 제어(Canonical/Non-canonical mode) 및 시간 측정 활용 |
|  **Q7**  | 프로세스 관련 함수 활용 연습 |  | `fork`, `exec`, `wait` 등을 이용한 프로세스 생성 및 관리 학습 |
|  **Q8**  | 시그널 관련 함수 활용 연습 |  | `signal`, `kill` 등을 이용한 시그널 처리 및 핸들러 구현 |
|  **Q9**  | 프로세스 간 통신(IPC) 함수 활용 연습 |  | **파이프(Pipe)**를 이용한 데이터 송수신 구현 |
|  **Q10**  | 메시지 큐를 이용한 채팅 프로그램 구현 |  | **메시지 큐(Message Queue)**를 활용한 비동기 통신 구현 |
|  **Q11**  | 공유 메모리(Shared Memory)를 이용한 파일 복사 프로그램 구현 |  | **공유 메모리**를 이용한 고속 IPC 및 동기화 이해 |
|  **Q12**  | DBUS를 이용한 서버-클라이언트 알림 프로그램 구현 |  | Linux 표준 IPC 시스템인 **D-Bus** 연동 학습 |

## 🛠️ 핵심 사용 기술 (Key Technologies)

* **언어:** Rust
* **시스템:** Linux (UNIX-like OS)
* **주요 개념:**
    * Rust 표준 라이브러리 (Vec, HashMap 등)
    * UNIX/Linux 시스템 호출 (File I/O, Process, Signal)
    * 프로세스 간 통신 (IPC: Pipe, Message Queue, Shared Memory)
    * 터미널 제어 및 D-Bus
